## Title
Battleship

## Description
A battleship game project to implement TDD (Test-Driven-Development) to testing the logic code. It includes two extra features, such as computer intelligent that can do adjacent hits after first hit and multiplayer feature (pass-device).

## Technologies
- HTML: for structure
- JavaScript: for logic and DOM manipulation
- CSS: for styling
- Git: for version control
- ES5 & ES6: for code organization
- Jest: for testing

## Note
Restart/Menu button is not implemented in this version, but the game core logic and UI are fully functional. Refresh the page to play the game again. This can be added in the later version.
